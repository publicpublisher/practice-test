# Colab-safe English notebook collection

All notebook filenames are URL-safe ASCII and include English descriptions.

Pattern:

```text
001_English-Tutorial-Title__English-Section-Title.ipynb
```

Korean titles and original filenames are preserved in `MANIFEST_COLAB_ENGLISH.json` and `COLAB_LINKS_MANIFEST.csv`.

## Colab URL template

```text
https://colab.research.google.com/github/<GITHUB_USER>/<REPO>/blob/<BRANCH>/<PATH>/<NOTEBOOK>.ipynb
```
